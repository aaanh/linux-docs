# Alacritty

Alacritty terminal emulator config file.

Tested on Linux-based Operating Systems.

For installation instructions, refer to <https://github.com/alacritty/alacritty>

Download and "install" my dotfile:

```sh
mkdir -p ~/.config/alacritty && wget --user-agent="chrome" https://linux.hoanganh.tech/_static/alacritty.yml -O ~/.config/alacritty/
```
