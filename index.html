ubuntu/debian first setup

  1. Update repository listing & Upgrade all existing packages
    sudo apt update && sudo apt upgrade
  2. apt packages
    sudo apt install git zsh net-tools build-essential powerline fonts-powerline fonts-firacode vim openssh-server tmux gnome-tweak-tool grub-customizer
  3. snap packages
    sudo snap install --classic code
    sudo snap install --classic zoom-client
    sudo snap install --classic discord
  4. zsh setup
    chsh -s /bin/zsh # change shell
    git clone https://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh # clone oh-my-zsh configs
    cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc # copy the config template into .zshrc
    vim ~/.zshrc # change ZSH_THEME="af-magic"
    git clone https://github.com/zsh-users/zsh-syntax-highlighting.git "$HOME/.zsh-syntax-highlighting" --depth 1 # syntax highlighting
    echo "source $HOME/.zsh-syntax-highlighting/zsh-syntax-highlighting.zsh" >> "$HOME/.zshrc" # source highlighting plugins on start
  5. Solid black 4K Resolution. No bullshit, no patterns.

tar

  • Create
    tar zcvf target.tar.gz sourcefile
  • Unpack
    tar zxvf sourcefile
  • List
    tar ztvf sourcefile

qemu

  1. Create Empty HDD Image:
    qemu-img create -f qcow2 name.img 69G
  2. Boot from CDROM/ISO Image:
    Linux-based:
    qemu-system-x86_64 -m 2048 -boot d -enable-kvm -smp 3 -net nic -net user -hda name.img -cdrom ~/path/to/iso/image.iso
    Mac:
    qemu-system-x86_64 -m 2048 -boot d -accel hvf -smp 3 -net nic -net user -hda name.img -cdrom ~/path/to/iso/image.iso
  3. Install chosen distro on blank HDD image
  4. Boot from HDD image
    qemu-system-x86_64 -m 2048 -boot d -enable-kvm -smp 3 -net nic -net user -hda name.img
    or
    qemu-system-x86_64 -m 2048 -boot d -accel hvf -smp 3 -net nic -net user -hda name.img
